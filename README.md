This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

A web app that finds the fairest London meeting point for a group, based on real TfL journey times rather than the geographic midpoint. Everyone adds their starting station, Rally checks live journey times across zones, and suggests the spot where nobody's commute is unfairly long, plus nearby venues to meet at.

Next.js · TypeScript · TfL API · Google Places API
